<h1>Tip Calculator</h1>

Bem-vindo à Calculadora de Gorjetas! Este é um projeto simples feito em JavaScript para ajudar você a calcular gorjetas com base na qualidade do serviço recebido. Basta informar o valor da conta e a qualidade do serviço, e a calculadora faz o resto!

🚀 Funcionalidades

Permite inserir o valor da conta.

Seleção de porcentagem de gorjeta com base na qualidade do serviço:

<li> Excelente: 10% </li>

<li> Ótimo: 8% </li>

<li> Bom: 5% </li>

<li> Ruim: 2% </li>
<br>
Exibe o valor da gorjeta calculada.

Exibe o valor total (conta + gorjeta).

🛠️ Tecnologias Utilizadas

Linguagem: JavaScript

HTML e CSS: Para a interface simples.
